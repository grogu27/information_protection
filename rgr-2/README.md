# 1. Постановка задачи
Необходимо реализовать протокол доказательства с нулевым знанием (ZKP) для задачи "Гамильтонов цикл".

Особенности задачи:
- Гамильтонов цикл предоставляется во входном файле.
- Программа не ищет цикл, а проверяет его с помощью протокола.
- Протокол демонстрирует доказательство без раскрытия информации о цикле.
- Для тестирования необходимо генерировать граф с корректным циклом с помощью дополнительной программы.

---

# 2. Структура проекта

Проект состоит из следующих элементов:

- `src/` — исходные файлы:
    - `main.cpp` — точка входа, загрузка графа и цикла, запуск ZKP.
    - `graph.hpp / graph.cpp` — работа с графом и циклом, чтение из файлов.
    - `zkp.hpp / zkp.cpp` — реализация протокола ZKP.
    - `gen.cpp` — генератор графа с корректным гамильтоновым циклом для тестирования.
- `build/` — скомпилированные объектные файлы.
- `Makefile` — сборка проекта, генератора, чистка, запуск всего проекта.
- `setup.sh` — скрипт для создания структуры и всех файлов проекта.
- `graph.txt` — файл с графом (создаётся генератором).
- `cycle.txt` — файл с гамильтоновым циклом (создаётся генератором).

```bash
.
├── build             # обьектные файлы
├── Makefile 
├── README.md 
├── setup.sh 
└── src
    ├── gen.cpp       # генератор графа с гамильтоновым циклом
    ├── graph.cpp     # реализация функций работы с графом
    ├── graph.hpp     # заголовок для graph.cpp
    ├── main.cpp      # точка входа, запуск ZKP
    ├── zkp.cpp       # реализация протокола доказательства с нулевым знанием
    └── zkp.hpp       # заголовок для zkp.cpp
```

# 3. Описание файлов

## main.cpp
- Загружает граф и цикл из файлов.
- Запускает протокол ZKP.
- Пункт задания: "Вне зависимости от варианта указывается дополнительная информация: последовательность вершин, задающая гамильтонов цикл".

## graph.hpp / graph.cpp
- Считывают граф и цикл из файлов.
- Пункт задания: "необходимо информацию о графах считывать из файла".

## zkp.hpp / zkp.cpp
- Реализация протокола доказательства с нулевым знанием:
    - Случайная перестановка вершин.
    - Коммит графа.
    - Challenge 0: проверка изоморфизма.
    - Challenge 1: проверка цикла.
- Пункт задания: "необходимо написать программу, реализующую протокол доказательства с нулевым знанием для задачи Гамильтонов цикл".

## gen.cpp
- Генератор графа с гарантированным гамильтоновым циклом.
- Пункт задания: "для тестирования необходимо будет генерировать правильные решения при помощи дополнительно разработанных программ".

## Makefile
- Сборка бинарника `zkp`.
- Сборка генератора `gen`.
- Цель `run` — автоматическая генерация файлов и запуск протокола.
- Цель `clean` — очистка бинарников и объектных файлов.

## setup.sh
- Создаёт структуру проекта, все исходники и Makefile.
