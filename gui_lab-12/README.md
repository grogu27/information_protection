# Лабораторная работа №12
## Тема: Ментальный покер

---

# 1. Постановка задачи
Реализовать алгоритм "Ментальный покер" для произвольного числа игроков и карт.
- Используем правила покера "Техасский холдем":
  - Каждому игроку раздать по 2 карты.
  - На стол выложить 5 карт.
- Консольная версия.
- Возможность добавить графический интерфейс в будущем.
- Обеспечить честность и секретность игры.

---

# 2. Теория
**Ментальный покер** — протокол честной раздачи карт без доверенного центра:
1. Колода карт представляется числами (0..51).  
2. Каждый игрок шифрует колоду своим ключом и перемешивает.  
3. Раздача карт осуществляется так, чтобы каждый видел только свои карты.  
4. Все игроки участвуют в формировании колоды → честность.  
5. Невозможно подглядывать чужие карты → секретность.

# 5. Защищённость и честность
- Каждая карта проходит через всех игроков → нельзя подделать раздачу.
- Только владелец карты видит её → секретность.
- Каждый игрок вносит случайность → честная игра.
